Part five: loops

Loops execute code over and over again either while a condition is fufilled (while loop) or for a certain number of times (for loop)


Here is a while loop that counts to ten

let x = 0;
while (x < 10) {
 x++
 console.log(x)
}

and here is a for loop that counts to ten

for (let x = 0; x < 10; x++) {
 console.log(x)
}
