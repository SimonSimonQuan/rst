Part eight: arrays

Arrays are ordered lists that hold multiple values in a single variable. They're like a shelf where each slot has a number (its index), and you can store or retrieve items from those slots. It also starts at 0 and counts up, so the first item has the index 0, the second has 1, etc.

here is how you make an array:

let colours = ["blue", "orange", "yellow"];

and here is how you can add something to it:

colours.push("green");

now we can look up a specific colour by labeling it's index in the list. We just added green so it would be fourth, where the index is 3

console.log(colours[3]);
>> green
