Part one: inputs and outputs

Inputs are data or commands that are entered into a program, and tell it what to do. Outputs are just the response of the program to these inputs.


For example, the console.log() function, which prints anyting you input, into the console, which is the output.

console.log("banana")
>> banana

Other common examples are alert(), prompt(), or buttons. Buttons can be used to execute code when clicked.
