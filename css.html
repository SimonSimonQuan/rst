Part three: css

CSS, which means Cascading Style Sheets is the language used to style and design the layout of a webpage. It controls things like colors, fonts, spacing, and layouts. Without CSS, websites would just be plain text and boxes.

here is some of the css used in this website, specifically the part that makes the highlighted boxes

.highlight {
    background-color: #eaf6ff;
    border-left: 6px solid #005f99;
    padding: 1rem;
    margin-top: 1rem;
}

it adds a background, a dark border on the left, and some padding around the edges.

for an example here is css used to make a fancy button

#fancy-button {
background: linear-gradient(135deg, #005f99, #ffa500);
color: white;
font-size: 1.1rem;
font-weight: bold;
padding: 0.75rem 2rem;
border: none;
border-radius: 50px;
box-shadow: 0 6px 12px rgba(0, 0, 0, 0.2);
cursor: pointer;
transition: all 0.4s ease;
letter-spacing: 1px;
position: relative;
overflow: hidden;
}
