Part two: variables

Variables can hold data, letting us reuse, change, and manage information easily throughout a program. It can be a number, text, or boolean which is true/false.


They are usually called like this:

let n = 5
var string = "meow meow meow meow"
const name = "scott"

let is only accesible within the block it was created in, maybe an if statement or something. var creates a more global varibale, and const is like a variable but you can't change it.


to assign a value to a variable, just write the variable and set it equal to something

n = 999
string = "aaaaa"

the following text box controls a variable which when set to true, will show a button for the next page. type in true to change the variable

function buttonAppear() {
 let textt = document.querySelector("#variable-box").value // this is the variable
 if (textt == "true"){ //checks if the variable is true
  document.querySelector("#variable-button").style.visibility = "visible"
 }
}
